## 🏥 Descripción

Sistema médico dental desarrollado para clínicas odontológicas, diseñado para gestionar pacientes, citas, historiales clínicos y tratamientos de forma eficiente.  
Permite centralizar la información médica y administrativa en una sola plataforma.

---

## 🚀 Características Principales

### 👨‍⚕️ Gestión de Pacientes
- Registro de pacientes  
- Historial clínico dental  
- Control de consultas  
- Seguimiento de tratamientos  

### 📅 Gestión de Citas
- Agenda de citas  
- Programación de consultas  
- Control de disponibilidad  

### 🦷 Área Odontológica
- Odontograma digital  
- Diagnósticos dentales  
- Registro de tratamientos (limpieza, extracción, ortodoncia, etc.)  
- Evolución del paciente  

### 💊 Tratamientos y Recetas
- Prescripciones médicas  
- Planes de tratamiento  
- Seguimiento clínico  

### 💼 Administración
- Gestión de usuarios (admin / doctor)  
- Control de pagos  
- Reportes del sistema  
- Gestión de base de datos  

---
